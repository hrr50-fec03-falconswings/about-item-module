## Usage

> Some usage instructions

### $ npm install

### $ npm run db:build
  - note: this will drop any local mysql db named 'products_db'

### $ npm run db:seed

### (Dev) In two separate terminals:
### $ npm run build (first terminal)
### $ npm start (second terminal)

### (Prod)
### $ webpack
### $ npm start

### login to AWS EC2 instance
### $ npm run aws
